User Stories

• As a user, when I visit the home page, I want to see a list of all the ingredients.
• As a user, when I visit an ingredient, I want to see a list of all the recipes that include the ingredient.
• As a user, when I visit a recipe, I want to see the recipe and related comments/reviews.
• As a user, I want to post comments on an existing recipe.
• As a user, I want to delete an existing comment.
• As a user, I want to upvote an existing recipe.
